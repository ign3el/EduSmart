# Product Context

## Project Overview
EduStory is an educational storytelling platform that generates personalized stories for children with AI-powered narration and interactive elements.

## User Goals
- Create engaging educational content for children
- Generate stories with customizable voices and characters
- Provide offline access to stories
- Enable multi-user management with admin capabilities

## Key Features
- AI story generation with Kokoro TTS integration
- Progressive Web App (PWA) for offline functionality
- User authentication and role management
- Admin dashboard for content management
- Voice customization and avatar selection
